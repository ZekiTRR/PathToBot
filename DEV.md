Структура
```text
src/
├── lib.rs               # Точка входа для DLL, макрос hudhook, инициализация
├── memory.rs            # Утилиты для поиска паттернов (AOB) и разбора инструкций
├── config.rs            # Хранилище строк-паттернов (сигнатур)
├── game/                # Слой абстракции над игрой (Твой SDK)
│   ├── mod.rs           # Реэкспорт модулей игры
│   ├── client.rs        # Твоя структура Client (базовый адрес, контейнер)
│   ├── player.rs        # Структура персонажа (ХП, мана, координаты)
│   └── offsets.rs       # Смещения внутри структур (если останутся динамические)
├── bot/                 # Логика автоматизации
│   ├── mod.rs           # Главный контроллер бота (вкл/выкл, логика)
│   └── actions.rs       # Функции взаимодействия (функция ходьбы, касты)
└── ui/                  # Графический интерфейс
    └── mod.rs           # Реализация RenderLoop для ImGui (отрисовка окна)
```

При старте программы, будет создаваться два потока, один отвечает за отрисовку UI
Второй отвечает за AOB скан. После скана, поток плавно превратиться в бесконечный цикл бота
