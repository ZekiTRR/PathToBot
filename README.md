# PathToBot

> **Проект в архиве.** Разработка на данный момент приостановлена, но может быть продолжена в будущем.

[English version](README_EN.md)

---

Бот для игры, написанный на Rust. Загружается как DLL в процесс игры.

## Возможности

- **AOB-скан** — поиск паттернов (сигнатур) в памяти процесса с поддержкой wildcard-байтов (`??`)
- **Mid-function хуки** — хуки прямо в середине функций через inline ASM; извлекают адреса напрямую из регистров (rcx, rax, rdi и т.д.), а не через цепочки указателей
- **MinHook** — перехват функций и подмена аргументов
- **Relay Cave** — аллокация памяти в радиусе ±2 ГБ для обхода ограничений x64 JMP

## Структура

```
src/
├── lib.rs            # Точка входа (DllMain)
├── memory.rs         # AOB-сканер, установка mid-function хуков, resolve RIP
├── config.rs         # Паттерны (сигнатуры)
├── game/
│   ├── client.rs     # Инициализация, установка хуков, чтение координат
│   ├── hooks.rs      # Naked-функции для mid-function хуков (inline ASM)
│   └── player.rs     # Структура персонажа
├── bot/
│   └── actions.rs    # Логика автоматизации
└── ui/
    └── mod.rs        # Интерфейс (не был реализован)
```

## Сборка

```bash
cargo build --release
```

Выходной файл: `target/release/PathToBot.dll`
